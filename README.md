# Крестики-нолики

Небольшой веб-проeкт, реализующий игру «Крестики-нолики» на чистом HTML, CSS и JavaScript. Проект легкий, без серверной части, предназначен для запуска прямо в браузере или на локальном статическом сервере.

## Стек
- Frontend: HTML, CSS, JavaScript
- Backend: нет
- База данных: нет

## Как запустить локально
1.Клонируйте репозиторий и перейдите в папку проекта.

2.В папке проекта запустите файл start.bat(в зависимости от вашего браузера), проект откроется в режиме инкогнито.
